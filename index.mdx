India Trade CLI is a free, open-source trading terminal for India's equity and derivatives markets. It runs seven AI analyst agents on any NSE or BSE listed stock, forces them into a structured bull-vs-bear debate, and outputs a fund-manager-style synthesis with trade plans across three risk profiles.

## Get started in four steps

<Steps>
  <Step title="Install">
    Clone the repo and install with pip:
    ```bash
    git clone https://github.com/hopit-ai/india-trade-cli.git
    cd india-trade-cli
    python -m venv .venv && source .venv/bin/activate
    pip install -e .
    ```
  </Step>
  <Step title="Set up an AI provider">
    Choose any supported provider. [Gemini](https://aistudio.google.com) has a free tier — no credit card needed. See [AI Providers](/ai-providers/overview) for all options.
  </Step>
  <Step title="Connect a broker (optional)">
    Run `trade --no-broker` to skip this step and use delayed market data. For real-time NSE/BSE data, connect [Fyers](/brokers/fyers) — it's free.
  </Step>
  <Step title="Start the REPL">
    ```bash
    trade
    ```
    Then run your first analysis:
    ```
    > analyze RELIANCE
    ```
  </Step>
</Steps>
